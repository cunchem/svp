
# Messenger application audit
## Context
Messenger applications have become one of the main means of communication. Those applications process personal data and are thus subject to associated regulation such as the GDPR. In order to assess the compliance of messenger application with the GDPR, a [standardised audit framework](https://www.edpb.europa.eu/our-work-tools/our-documents/support-pool-expert-projects/standardised-messenger-audit_en) has been created.

## Objective
The objective of this project is to perform an evaluation of a messenger application using the standardised audit methodology. Once a messenger application has been selected, apply the methodology described in the document entitled [Audit methodology](https://www.edpb.europa.eu/system/files/2024-06/standardisedmessengeraudit_d2audit-methodology_edpb-spe-programme_en.pdf) and report the results of the evaluation in an evaluation sheet.
